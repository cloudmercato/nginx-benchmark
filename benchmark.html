Nginx benchmark

Nginx benchmark display a set of endpoint helping you to test your reverse-proxy

The endpoint aren't analysis tool in themselves but must be combined with the HTTP client of your choice.

/download

Generate a file and download it

Try it
Total size of the file


Size of each chunk in the download stream 

curl http://${host}/download?size=20480&chunk_size=2048

/urandom

Generate a file from /dev/urandom, just to stress CPU.

Try it 

curl http://${host}/urandom?size=2048

/zero

Generate a file from /dev/zero with a lower stress than urandom



curl http://${host}/zero?size=1

/sleep

Just wait x seconds



curl http://${host}/sleep?time=1
